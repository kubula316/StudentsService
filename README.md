Część aplikacji webowej kursów dla studentów:

Serwis ten odpowiedzialny jest za zarządzanie studentami oraz ich autoryzacje za pomocą tokenu JWT.
Wszystkie endpointy są zabezpieczenone(oprócz endpointów rejestracji i logowania) i wymagają autoryzacji tokenem JWT w serwisie Student.

Do działania serwisu trzeba zaimplementować zmienne środowiskowe znajdujące się w pliku .env
